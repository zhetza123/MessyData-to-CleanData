# 🧹 AI-Powered Procurement Data Cleaning & Analysis

## 📌 Project Overview
This project demonstrates the transition from raw, "messy" procurement data to a structured, business-ready dashboard. Using **Microsoft Excel Copilot (AI)** and advanced data processing techniques, I automated the cleaning of 50+ inconsistent records—standardizing supplier names, converting currencies, and removing duplicates.

This is a real-world simulation of the data challenges faced in professional procurement and supply chain environments.

## 🛠 Features & Workflow
1. **Raw Data Ingestion:** Handled a dataset with inconsistent date formats, mixed currencies (EUR/RSD), and fragmented supplier names (e.g., "Grindex", "grindex doo", "GRINDEX").
2. **AI-Driven Cleaning (Copilot):**
   - **Standardization:** Automatically grouped variations of supplier names into single entities.
   - **Currency Normalization:** Used AI prompts to identify EUR values and convert them to RSD using a specific exchange rate.
   - **Data Integrity:** Identified and removed duplicate entries based on fuzzy matching logic.
3. **Data Visualization:** Built an interactive dashboard featuring **PivotTables**, **Slicers**, and **PivotCharts** for instant cost analysis.

## 🚀 Why this matters
In modern procurement, data is often fragmented across different systems. This project proves that:
- **Efficiency:** AI can reduce hours of manual cleaning to minutes.
- **Accuracy:** Automated standardization prevents double payments and reporting errors.
- **Decision Making:** Interactive slicers allow management to track spending by supplier or status in real-time.

## 🧰 Technologies Used
- **Microsoft Excel** (PivotTables, Slicers, Data Modeling)
- **Microsoft 365 Copilot** (AI Prompt Engineering for Data Cleaning)
- **Financial Logic** (Currency conversion and procurement workflow)
